# Web Security Scripts

This repo contains automated exploit scripts and custom tools.
The main goal is to turn the tedious manual exploitation process into automated scripts.

## Structure

* **`1_PortLab_PoCs`**: PoC scripts; each script targets a different scenario.
* **`2_SQLi_Enum_Framework`**: An OOP-based automated enumeration framework for Blind SQL Injection.

## Tech Stack
* Python 3

## Disclaimer
All scripts in this repository are for educational purposes and authorized penetration testing (e.g., PortSwigger Web Security Academy) only. Please do not use them against any unauthorized targets.
